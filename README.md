# Distributed Systems Abstractions
This an experimental repository with implementations of different abstractions discussed in Distributed Systems theory.
The components and their algorithms are based on the contents of the book _"Introduction to Secure and Reliable Distributed Programming"_

# How to Use
1. Modify `process.c` to include and use any of the implemented abstractions.
2. Run this command `make stacking_components`
3. Run the executable with `./stacking_components`
